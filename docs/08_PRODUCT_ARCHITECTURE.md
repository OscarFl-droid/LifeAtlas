# Product Architecture

## Seven primary systems
1. Local event archive
2. Longitudinal state engine
3. Evidence and recommendation registry
4. Sequence and threshold engine
5. Living Atlas renderer
6. Offline application and synchronisation layer
7. Domain and audience template platform

## Supporting systems
Consent and privacy, correction and deletion, model versioning, encrypted backup, evidence-review workflow, accessibility, permissions, audit log, deterministic replay, export, user-controlled telemetry.

## Separation of concerns
The user records ordinary life. Domain experts define what matters. The engine identifies how history changes what should happen next.
