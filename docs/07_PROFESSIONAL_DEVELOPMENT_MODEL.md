# Professional Development Model

Professional development is one domain, not the whole product.

```text
Current profession
+ current role
+ desired role
+ aspirations
+ demonstrated behaviours
+ constraints
+ history
→ capability map
→ gaps
→ next actions
```

Broad branches: execution, communication, collaboration, judgement, leadership, learning, adaptability, organisation, strategic thinking, domain expertise, professional visibility, career transition readiness.

Completing a course is exposure. Applying it, seeking feedback, adjusting behaviour, and supporting others provides evidence of capability.

Recommendations should be concrete and tailored, not generic.
